Methods

Live Validator will recognize the following methods

setRequired

Parameters: {boolean} <doCheck = false>

This will mark the input(s) as required. The theme should take care of any visuals that it will use to indicate that the input is required. You can specify if a check should be done to see if the input(s) is/are currently empty at the same time (false by default), else this will happen automatically when the input is changed.

unsetRequired

Parameters: None

Use this to make the input(s) unrequired. Like setRequired, the theme should handle the removal of the visual mark which indicated that the input was required. A check is automatically run with this call to remove any visuals that may no longer be needed.

Example of changing required state

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <input type="text" id="changeRequired"/>
    <button type="button" class="uk-button" id="btn-setRequired">Set Required</button>
    <button type="button" class="uk-button" id="btn-unsetRequired">Unset Required</button>
</div>

  • $( function() {
    var input = $( '#changeRequired' ).LiveValidator();

    $( '#btn-setRequired' ).click( function( e ) {
        $( input ).LiveValidator( 'setRequired' );
    } );
    $( '#btn-unsetRequired' ).click( function( e ) {
        $( input ).LiveValidator( 'unsetRequired' );
    } );
} );
Notice how the input is marked as missing when setRequired is clicked because it has true for the doCheck option. Also notice how unsetRequired automatically removes the missing visual if needed.

enableLive

Parameters: {boolean} <doCheck = false>

This turns live validation on - validation that happens as the user types input. It cause all the checks set in checks to be checked on each input event. Here the option doCheck can also be used to indicate that a full check should be run after live validation has been enabled.

disableLive

Parameters: None

Use this if you want to disable the live validation. This will cause the set checks to only be run when the input looses focus.

Example of changing live validation state

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <input type="text" id="changeLiveValidation" pattern="^[0-9]+$" title="Only numerics allowed"/>
    <button type="button" class="uk-button" id="btn-enableLive">Enable Live Validation</button>
    <button type="button" class="uk-button" id="btn-disableLive">Disable Live Validation</button>
</div>

  • $( function() {
    var input = $( '#changeLiveValidation' ).LiveValidator();

    $( '#btn-enableLive' ).click( function( e ) {
        $( input ).LiveValidator( 'enableLive' );
    } );
    $( '#btn-disableLive' ).click( function( e ) {
        $( input ).LiveValidator( 'disableLive' );
    } );
} );

addChecks

Parameters: {array} checks

This method allows you to add a check or multiple checks after LiveValidator has been instantiated. This can be useful when you want to change the validation based on other inputs.

removeChecks

Parameters: {array} checks

It is also possible to remove checks as is needed after instantiation.

removeAllChecks

Parameters: None

You can also clear all checks and start fresh using this.

Example of changing the checks

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <input type="text" id="changeChecks" required/>
    <button type="button" class="uk-button" id="btn-addCheck">Add a Check</button>
    <button type="button" class="uk-button" id="btn-removeLast">Remove Last Added</button>
    <button type="button" class="uk-button" id="btn-removeAll">Remove All Checks</button>
</div>

  • $( function() {
    var input = $( '#changeChecks' ).LiveValidator();

    var checks = [
        "isNumber",
        {"min": 5},
        {"max": 15}
    ];
    var i = 0;

    $( '#btn-addCheck' ).click( function( e ) {
        if ( i < checks.length )
            $( input ).LiveValidator( 'addChecks', checks[i++] );
    } );
    $( '#btn-removeLast' ).click( function( e ) {
        if ( i > 0 )
            $( input ).LiveValidator( 'removeChecks', checks[--i] );
    } );
    $( '#btn-removeAll' ).click( function( e ) {
        i = 0;
        $( input ).LiveValidator( 'removeAllChecks' );
    } );
} );

isValid

Parameters: None

This can be used to check if an input is valid. It will rerun the required check and then the other testers if needed.
When called using jQuery it will run on all inputs in the selector and on all children of the selector(s).

destroy

Parameters: None

Use this when you want to destroy the plug-in.
