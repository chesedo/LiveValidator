Getting Started

These are the basic steps to get started with the plug-in

Creation

To start using the plug-in, just call it on a jQuery object.

Calling the plug-in

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <input type="text" id="auto-required" required />
</div>

  • $( function() {
    test = $( '#auto-required' ).LiveValidator();
} );
Notice how the required attribute is detected on the input and handled by the theme to add the asterisk.

Passing options to the plug-in

It is also possible to overwrite the plug-in options per instantiation. Just pass in a object with any of the valid options.

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <input type="text" id="passed-options"/>
</div>

  • $( function() {
    $( '#passed-options' ).LiveValidator( { required: true } );
} );

Passing options via data

The plug-in will also check for data-* attributes and combines them into the options. This makes it easy to have a lot of inputs with many options.

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <!-- The `isNumber` check comes from html5validation.js -->
    <input type="text" id="data-options" data-checks='[ "isNumber" ]'/>
</div>

  • $( function() {
    $( '#data-options' ).LiveValidator();
} );
Note the example of how to set array & object options correctly - with the double quotes on the inside - this insures that they are parsed correctly.

Changing default options

Sometimes it is also needed to change the default options before initiating a set of inputs - setting the theme data is a good example of this.

To do this, just call the plug-in directly and pass in a object of options to overwrite.

  • Example
  • HTML
  • JavaScript
  • <div id="numbers">
    <div class="row">
        <label>Label</label>
        <input type="text" id="default-options1"/>
    </div>
    <div class="row">
        <label>Label</label>
        <input type="text" id="default-options2"/>
    </div>
</div>

  • $( function() {
    // The `isNumber` check comes from html5validation.js
    $.LiveValidator( { checks: [ 'isNumber' ] } );

    // Inputs will contain the two inputs and not `div#numbers`
    var inputs = $( '#numbers' ).LiveValidator();
} );

Also note how to instantiate multiple inputs with one call - which can be an alternative to overwriting the defaults. This is because the plug-in automatically filters the jQuery object to contain only supported inputs. They are also returned from the instantiation.

After creation

Now that you know how to create a instance of the plug-in on inputs, you will be able to do two primary things with this instance.

Calling methods

The plug-in exposes some methods that can be called after instantiation. They can be called by passing them in as a string when calling the plug-in.

  • Example
  • HTML
  • JavaScript
  • <div class="row">
    <label>Label</label>
    <input type="text" id="changeRequired"/>
    <button type="button" class="uk-button" id="btn-setRequired">Set Required</button>
    <button type="button" class="uk-button" id="btn-unsetRequired">Unset Required</button>
</div>

  • $( function() {
    var input = $( '#changeRequired' ).LiveValidator();

    $( '#btn-setRequired' ).click( function( e ) {
        $( input ).LiveValidator( 'setRequired' );
    } );
    $( '#btn-unsetRequired' ).click( function( e ) {
        $( input ).LiveValidator( 'unsetRequired' );
    } );
} );

The mighty isValid method

The most important method is the isValid method. It will check if the jQuery object(s) - which is filtered to inputs only - is valid and passes all their constraints.

  • Example
  • HTML
  • JavaScript
  • Multiple isValid checking

    Check all 
  • <fieldset id="fieldset">
    <legend>Multiple isValid checking</legend>
    <div class="row">
        <label>Label</label>
        <input type="text" id="isValid1" required/>
    </div>
    <div class="row">
        <label>Label</label>
        <input type="text" id="isValid2"/>
    </div>
</fieldset>
<p id="results"> </p>
<a class="uk-button uk-margin-top" id="isValid">Check all</a>

  • $( function() {
    var $result = $( '#results' );

    $( '#fieldset' ).LiveValidator( { checks:[ "isNumber" ] } );

    $( '#isValid' ).click( function() {
        // It will filter the #fieldset's inputs only
        var isValid = $( '#fieldset' ).LiveValidator( 'isValid' );
        if ( isValid ) {
            $result.text( 'Valid' )
                .addClass( 'uk-text-success' )
                .removeClass( 'uk-text-warning' );
        } else {
            $result.text( 'Invalid' )
                .addClass( 'uk-text-warning' )
                .removeClass( 'uk-text-success' );
        }
    } );
} );
Note how the inputs with errors are marked - automatically by this method - using the theme to guide the user in fixing them.
